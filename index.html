Hurra!

Soldier welcome to LoadoutGenerator. Here you will be able to generate some random loadouts for Battlefield 4 to mix things up and have some fun trying new ways to play.

This little project is made possible thanks to the bf4stats API that provides you the hability to generate a random loadout with YOUR unlocks.

This started as a side project to refresh my Battlefield 4 play times. While searching for a random generator on the internet I found nothing customized for me.

I then took the initiative to use my skills in web developement to create a fully customized loadout generator for BF4.

I really want to take this project further by including support for other games like Battlefront.

Now go have fun in the Battlefield !

Please feel free to contact me at matth.couvreur@gmail.com for any kind of feedback and request.
