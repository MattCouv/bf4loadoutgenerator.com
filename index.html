Welcome Soldier, to LoadoutGenerator. Here you will be able to generate some random loadouts for Battlefield 4 to mix things up and have some fun trying new things.

This little project is made possible thanks to the bf4stats API that provides you the hability to generate a random loadout with YOUR unlocks.

This started as a side project too refresh my Battlefield 4 play times. While searching for a random generator on internet i found nothing customized for me.

I then took the initiative to use my skills in web developement to create a fully customize loadout generator for BF4.

I realy went to take this project further including support for other games like Battlefront.

Now go have fun in the Battlefield
